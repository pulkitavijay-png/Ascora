# Bill of Materials (BOM) - Ascora V1.0

This document outlines the hardware components used in the Ascora mobile educational unit. The build is designed for high-performance AI processing at the edge using the Raspberry Pi 5 ecosystem.

## 1. Core Computing & Intelligence
| Component | Specification | Purpose |
| :--- | :--- | :--- |
| **Processor** | Raspberry Pi 5 (8GB RAM) | Main AI orchestrator and logic unit |
| **Cooling** | Active Cooler for Raspberry Pi 5 | Maintains performance during long outdoor sessions |
| **Storage** | 128GB MicroSD Card (Class 10) | OS and local edge-caching of lessons |

## 2. Interaction & Display
| Component | Specification | Purpose |
| :--- | :--- | :--- |
| **Primary Screen** | 10.1" IPS LCD Touchscreen | Facilitator Dashboard and AI Character display |
| **Visual Output** | Vertical Mount LED Strip | Expressive facial cues and status indicators |
| **Audio** | Dual 5W Internal Speakers | Clear AI voice output for group learning |

## 3. Power & Connectivity
| Component | Specification | Purpose |
| :--- | :--- | :--- |
| **Power Bank** | 20,000mAh PD (65W Output) | Portable power for 8+ hours of usage |
| **Network** | 4G/LTE USB Dongle | Real-time connection to Gemini 1.5/3.1 API |
| **Cabling** | High-speed HDMI & USB-C | Internal component data & power flow |

## 4. Structural Chassis
| Component | Specification | Purpose |
| :--- | :--- | :--- |
| **Base** | 280mm x 300mm Weighted Base | Stability for the vertical totem structure |
| **Mobility** | 4x Industrial Caster Wheels | Easy transport across park terrain |
| **Body Shell** | UV-Resistant Acrylic/PVC | Weather-hardy casing for internal electronics |
